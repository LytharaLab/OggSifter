# OggSifter

Инструмент для извлечения и фильтрации аудиокэша Roblox - легко извлекайте аудиофайлы OGG из кэша Roblox.

## Возможности

- **Интеллектуальное сканирование**: Рекурсивно сканирует указанные папки и автоматически определяет файлы, содержащие аудио OGG
- **Обнаружение магического числа**: Находит магическое число "OggS" в файлах и извлекает действительное аудио OGG
- **Встроенный плеер**: Воспроизводит извлеченное аудио прямо в программе без дополнительного программного обеспечения
- **Фильтрация и сохранение**: Можно прослушивать и сохранять только нужные аудиофайлы

## Системные требования

- Операционная система Windows
- Среда выполнения .NET 9.0

## Использование

1. **Выберите исходную папку**: По умолчанию автоматически обнаруживает каталог кэша Roblox (`%USERPROFILE%\AppData\Local\Roblox\rbx-storage`), или можно вручную выбрать другую папку
2. **Выберите каталог сохранения**: Выберите место, где вы хотите сохранить конечные аудиофайлы
3. **Сканировать и извлечь**: Нажмите кнопку "Сканировать и извлечь" для начала сканирования
4. **Прослушивание и фильтрация**:
   - Используйте "Предыдущий" и "Следующий" для просмотра извлеченного аудио
   - Нажмите "Воспроизвести" для прослушивания текущего аудио
   - Если вы удовлетворены, нажмите "Сохранить это аудио" для сохранения в указанный каталог
5. **Управление временными файлами**: Нажмите "Открыть временный каталог" для просмотра всех извлеченных временных файлов

## Сборка проекта

```bash
# Клонировать репозиторий
git clone https://github.com/LytharaLab/OggSifter.git

# Перейти в каталог проекта
cd OggSifter

# Восстановить пакеты NuGet
dotnet restore

# Собрать проект
dotnet build

# Запустить проект
dotnet run --project OggSifter
```
