# OggSifter

Robloxオーディオキャッシュ抽出・フィルタリングツール - RobloxキャッシュからOGGオーディオファイルを簡単に抽出できます。

## 機能

- **スマートスキャン**: 指定したフォルダを再帰的にスキャンし、OGGオーディオを含むファイルを自動的に識別
- **マジックナンバー検出**: ファイルから「OggS」マジックナンバーを見つけ、有効なOGGオーディオを抽出
- **内蔵プレーヤー**: 追加のソフトウェアなしで、プログラム内で直接抽出したオーディオを再生
- **フィルタリングと保存**: 必要なオーディオファイルだけを聴いて保存できます

## システム要件

- Windows オペレーティングシステム
- .NET 9.0 ランタイム

## 使用方法

1. **ソースフォルダの選択**: デフォルトではRobloxキャッシュディレクトリ（`%USERPROFILE%\AppData\Local\Roblox\rbx-storage`）を自動的に検出します。他のフォルダを手動で選択することもできます
2. **保存先ディレクトリの選択**: 最終的なオーディオファイルを保存する場所を選択します
3. **スキャンと抽出**: 「スキャンして抽出」ボタンをクリックしてスキャンを開始します
4. **プレビューとフィルタリング**:
   - 「前へ」と「次へ」を使用して抽出したオーディオを閲覧
   - 「再生」をクリックして現在のオーディオをプレビュー
   - 満足したら「このオーディオを保存」をクリックして指定したディレクトリに保存
5. **一時ファイルの管理**: 「一時ディレクトリを開く」をクリックすると、抽出したすべての一時ファイルを表示できます

## プロジェクトのビルド

```bash
# リポジトリをクローン
git clone https://github.com/LytharaLab/OggSifter.git

# プロジェクトディレクトリに移動
cd OggSifter

# NuGetパッケージを復元
dotnet restore

# プロジェクトをビルド
dotnet build

# プロジェクトを実行
dotnet run --project OggSifter
```
